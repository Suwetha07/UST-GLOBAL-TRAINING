# Linux Scripting Assignments - UST Global Training

This repository contains 7 Linux scripting assignments with their code and output screenshots.

1. Number Classification (if + function)
2. Factorial of a Number (while + function)
3. Sum of N Numbers (for + function)
4. Prime Number Check (if + for + function)
5. Reverse a Number (while + if)
6. File Existence & Permission Check (if + function)
7. Menu-Driven Calculator (while + case + function)

Each folder contains:
- The shell script (`*_script.sh`)
- Nano editor screenshot (`*_nano.png`)
- Terminal output screenshot (`*_output.png`)
